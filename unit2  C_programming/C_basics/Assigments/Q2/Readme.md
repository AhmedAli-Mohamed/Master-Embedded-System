# Ex2 :
Write C Program to Print a Integer Entered by a User

i should see the Console as following:

##########Console-output###

Enter a integer: 25

You entered: 25

# Solution

```c
#include <stdio.h>
void main()
{
    int x=0;
    printf("Enter a integer : ");
    scanf("%d",&x);
    printf("You entered : %d ",x);

    
    
}


```
